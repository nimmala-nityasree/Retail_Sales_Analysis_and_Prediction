# Retail Sales Analysis and Prediction Using Machine Learning

## Overview

This project performs an end-to-end data science analysis on a retail sales dataset. The objective is to analyze sales patterns, identify key business insights, and build a machine learning model to predict sales based on historical data.

The project demonstrates essential data science skills, including data cleaning, exploratory data analysis (EDA), visualization, feature engineering, machine learning, and model evaluation.

## Objectives

* Analyze retail sales data to uncover business trends.
* Identify factors influencing sales and profit.
* Visualize sales performance across categories and regions.
* Build a machine learning model to predict sales.
* Generate actionable business insights from the data.

## Dataset

Dataset: Sample Superstore Dataset

The dataset contains information about:

* Orders
* Sales
* Profit
* Quantity
* Discounts
* Product Categories
* Customer Segments
* Regions

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Project Workflow

### 1. Data Collection

* Imported the retail sales dataset.
* Loaded data into a Pandas DataFrame.

### 2. Data Cleaning

* Checked for missing values.
* Removed duplicate records.
* Verified data types and consistency.

### 3. Exploratory Data Analysis (EDA)

* Analyzed sales distribution.
* Examined category-wise sales performance.
* Compared regional sales and profits.
* Investigated relationships between key variables.

### 4. Data Visualization

Created visualizations including:

* Sales by Category
* Profit by Category
* Sales by Region
* Correlation Heatmap
* Feature Importance Chart

### 5. Machine Learning

Implemented a Random Forest Regressor to predict sales using:

* Quantity
* Discount
* Profit

### 6. Model Evaluation

Evaluated model performance using:

* Mean Absolute Error (MAE)
* R² Score

## Key Findings

* Technology products generated the highest sales revenue.
* Sales performance varied across different regions.
* Discounts had a significant impact on profitability.
* Quantity and profit were important factors influencing sales.
* The machine learning model achieved effective sales prediction performance.

## Business Impact

The insights generated from this analysis can help retail businesses:

* Improve inventory planning.
* Optimize discount strategies.
* Focus on high-performing product categories.
* Forecast future sales more effectively.
